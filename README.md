# 📘 SkillSphere

## 🚀 Project Name
SkillSphere

---

## 🎯 Project Purpose
SkillSphere is an online learning platform where users can explore courses, view course details, and learn modern in-demand skills.  
The project is built to practice full-stack development using Next.js, authentication, and modern UI design.

---

## 🌐 Live URL
https://a08-skill-sphere.vercel.app/  

---

## ✨ Key Features

- 🔍 Search courses by title
- 📚 View all courses with details page
- ⭐ Popular courses (top-rated system)
- 👨‍🏫 Top instructors slider
- 🔐 Authentication (Register / Login / Logout)
- 👤 My Profile page with update feature (name & image)
- 🎨 Modern UI with purple theme
- 📱 Fully responsive design
- ⏳ Loading skeletons / Suspense loading UI
- 🔔 Toast notifications for success & error messages
- 🧭 Sticky navbar with active route highlight

---

## 📦 Installation & Setup

```bash
git clone https://github.com/jumana-rahman/A08-SkillSphere.git
cd a08-skillsphere
npm install
npm run dev

```

## 🏗️ Build for Production

```bash
npm run build

```
